## Getting Started

To run this quickstart you can fork and clone this repo.

This webapp only work for Baltic LUKE project.

To run the application:

```bash
# Install the dependencies
bower install

# Install simple web server
npm install -g serve

# Run
serve
open localhost on port 3000
```
